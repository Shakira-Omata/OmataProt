# SalamaHub: SRHR Digital Platform

SalamaHub is a youth-friendly, inclusive, and judgment-free sexual and reproductive health and rights (SRHR) platform for young people in Kenya. It intentionally includes young people with disabilities and marginalized groups.

## Key Features

- **Inclusive Design**: Built for accessibility from the ground up.
- **Accessibility Panel**: Real-time font size scaling and high-contrast mode toggling.
- **Kenya-Focused Content**: 100+ entries covering SRHR topics, rights, myths, and support services tailored for the Kenyan context.
- **Support Directory**: Direct access to verified help clinics and mental health resources.
- **Searchable Education**: Easily find information in our comprehensive SRHR topic library.

## Getting Started

To run the platform locally, follow these steps:

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Open in Browser**
   Navigate to `http://localhost:5173` to view the platform.

## Accessibility Controls

- Use the **Sidebar** to toggle between text sizes (Small, Base, Large, XL).
- Enable **High Contrast Mode** for improved visibility.
- The platform is designed to be fully navigable via keyboard.

## Project Structure

- `src/data/`: JSON mock data for all sections.
- `src/context/`: Accessibility management.
- `src/components/`: Reusable UI elements (Layout, Navbar, Sidebar).
- `src/pages/`: Individual view components.
